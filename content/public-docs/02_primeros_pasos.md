# Primeros pasos

Esta guía explica cómo empezar a usar SprintRoom desde cero.

## Crear una cuenta

1. Abre SprintRoom.
2. Haz clic en **Crear cuenta gratis**.
3. Ingresa tu nombre completo.
4. Ingresa tu correo electrónico.
5. Crea una contraseña de mínimo 8 caracteres.
6. Haz clic en **Crear cuenta**.
7. Después del registro, SprintRoom te redirige al inicio de sesión.

También puedes registrarte con Google usando **Registrarme con Google**.

## Iniciar sesión

1. Abre la pantalla de inicio de sesión.
2. Usa una de estas opciones: **Continuar con Google** o correo electrónico y contraseña.
3. Haz clic en **Entrar**.
4. Al iniciar sesión, SprintRoom abre el dashboard.

## Navegación principal

Después de iniciar sesión verás el menú principal:

| Sección | Uso |
|---|---|
| Dashboard | Resumen de proyectos visibles, tareas personales y avance promedio. |
| Imagina | Asistente de IA para convertir una idea en un proyecto con historias y tareas. |
| Proyectos | Crear proyectos y abrir espacios de trabajo. |
| Mis tareas | Consultar tareas asignadas a tu usuario. |
| Docs | Leer la documentación pública dentro de la aplicación. |
| Cuenta | Editar perfil, ver tu usuario ID y cerrar sesión. |

En dispositivos móviles, abre el menú lateral con el botón de menú superior.

## Actualizar tu perfil

1. Entra a **Cuenta**.
2. Edita tu nombre completo o correo electrónico.
3. Haz clic en **Guardar perfil**.

En esta pantalla también verás:

| Campo | Descripción |
|---|---|
| Usuario ID | Identificador que puedes compartir para que te agreguen a proyectos. |
| Rol global | `Member` o `Administrator`. |
| Origen | Forma en que fue creada tu cuenta. |

## Compartir tu Usuario ID

Para que otra persona te agregue a un proyecto:

1. Entra a **Cuenta**.
2. Copia tu **Usuario ID**.
3. Envíalo al propietario o mantenedor del proyecto.
4. Esa persona podrá agregarte desde la sección **Miembros** del proyecto.

## Crear tu primer proyecto

1. Entra a **Proyectos**.
2. Completa **Nombre**.
3. Opcionalmente agrega **Descripción**.
4. Opcionalmente agrega **Referencia externa**.
5. Haz clic en **Crear proyecto**.

Al crear el proyecto:

- Quedas como propietario.
- El proyecto inicia con 0% de avance.
- Puedes agregar miembros, historias y claves MCP desde el detalle del proyecto.

## Crear la primera historia

1. Abre un proyecto.
2. Busca la sección **Historias de usuario**.
3. Escribe el título de la historia.
4. Agrega una descripción si aplica.
5. Haz clic en **Crear historia**.

SprintRoom abre el tablero de la historia creada para que puedas crear tareas.

## Crear una tarea

1. Abre una historia de usuario.
2. Haz clic en **Nueva tarea**.
3. Escribe el título de la tarea.
4. Agrega una descripción si aplica.
5. Selecciona miembros asignados si aplica.
6. Haz clic en **Crear tarea**.

La tarea se crea inicialmente dentro del tablero de la historia.

## Mover una tarea

Puedes mover una tarea de dos formas:

1. Arrastrándola entre columnas del tablero Kanban.
2. Abriendo **Ver detalle** y seleccionando un estado manualmente.

Los estados disponibles son:

- Sin Empezar.
- En Desarrollo.
- Probando.
- En Revisión.
- Completada.

## Agregar comentarios

1. Abre una tarea con **Ver detalle**.
2. Escribe un comentario de seguimiento.
3. Haz clic en **Agregar comentario**.

Los comentarios son inmutables: no se editan ni se eliminan desde la interfaz actual.

## Eliminar elementos

SprintRoom exige confirmación explícita para evitar eliminaciones accidentales.

| Elemento | Confirmación requerida |
|---|---|
| Tarea | Escribir exactamente el título de la tarea. |
| Historia | Escribir exactamente el título y luego `ELIMINAR TODO`. |
| Proyecto | Escribir exactamente el nombre y luego `ELIMINAR TODO`. |
| Clave MCP | Escribir exactamente la descripción de la clave. |

Eliminar una historia también elimina sus tareas. Eliminar un proyecto elimina sus historias y tareas relacionadas.
