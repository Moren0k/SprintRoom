# Imagina: asistente para planear proyectos

**Imagina** es el asistente de SprintRoom para convertir una idea en un proyecto estructurado con historias de usuario y tareas.

## Qué puedes hacer

Con Imagina puedes:

- Describir una idea de producto o iniciativa.
- Conversar con el asistente para aclarar alcance.
- Generar un plan de proyecto.
- Revisar historias y tareas propuestas.
- Crear el proyecto completo en SprintRoom.

## Flujo recomendado

1. Entra a **Imagina**.
2. Describe tu idea de proyecto.
3. Responde las preguntas del asistente.
4. Cuando el contexto sea suficiente, haz clic en **Generar plan**.
5. Revisa el plan generado.
6. Si necesitas ajustes, haz clic en **Seguir mejorando**.
7. Si el plan está listo, haz clic en **Planear proyecto**.

## Qué contiene el plan

| Elemento | Descripción |
|---|---|
| Nombre del proyecto | Nombre sugerido para el proyecto. |
| Descripción | Resumen del objetivo y alcance. |
| Referencia externa | Referencia opcional. |
| Historias de usuario | Funcionalidades o capacidades principales. |
| Tareas | Trabajo ejecutable dentro de cada historia. |

## Crear el proyecto desde el plan

Al hacer clic en **Planear proyecto**, SprintRoom crea un proyecto, sus historias de usuario y las tareas de cada historia. Después te redirige al detalle del proyecto creado.

## Buenas prácticas para describir tu idea

Incluye información como el problema a resolver, usuarios, funcionalidades imprescindibles, restricciones, resultado esperado al final del sprint y tareas técnicas conocidas.

Ejemplo:

```text
Quiero crear una plataforma interna para registrar solicitudes de soporte.
La usarán empleados y agentes de soporte.
Necesitamos crear solicitudes, asignarlas, comentar avances y cerrar casos.
El primer sprint debe cubrir autenticación, listado de casos y tablero de estados.
```

## Limitaciones

Imagina ayuda a planear, pero debes revisar el resultado antes de crear el proyecto. La creación final ocurre solo cuando confirmas el plan.
